## 编译
### 开始之前
你应确保您已安装以下工具

- [x] i686-elf-gnu 12.1.0 及以上版本
- [x] grub 2.06 及以上版本 

### 编译
如果你想要使用GRUB 2.06引导，则执行以下指令

```
make build-grub2
```
